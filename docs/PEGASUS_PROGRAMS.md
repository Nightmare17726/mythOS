# mythOS Pegasus Edition - Complete Program List

**Target Size**: 150MB (increased from 85MB for full features)
**Focus**: Accessibility, giant UI, voice control, elderly-friendly

## Giant UI Mode Configuration
- **XFCE4** - Lightweight desktop (instead of just Openbox)
- **Custom theme** - Extra-large icons (192px)
- **Huge fonts** - 24pt minimum system-wide
- **High contrast** - Multiple color schemes
- **Large cursor** - 64px mouse pointer
- **Touch-optimized** - All buttons 80px+ height

## Voice Control & Accessibility
- **Mycroft AI** - Open-source voice assistant
  - Wake word: "Hey Mycroft"
  - Commands: "Open browser", "Read email", "What time is it"
- **eSpeak NG** - Text-to-speech engine
- **Speech Dispatcher** - TTS coordination
- **pocketsphinx** - Offline speech recognition
- **simon** - KDE speech recognition
- **Orca** - Screen reader with voice
- **GOK** - On-screen keyboard (giant keys)
- **Florence** - Virtual keyboard
